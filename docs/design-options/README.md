# Design options

Three self-contained HTML mockups exploring the visual direction for the auction platform. Open any file directly in a browser - no build step, no dependencies.

## Status

Decision pending with the product owner. Until then, styling stays behind a swappable CSS-variable token layer so any candidate (or a blend) can be adopted without touching component markup.

## Points of overlap

All three candidates already agree on:

- Dark 16:9 TV boards, designed at 1600x900 and scaled to fit
- Per-lot club re-dress of the board
- Crest plus player photo treatment
- Green/red/amber value semantics
- Tabular numerals throughout
- The sealed reveal as the signature moment
